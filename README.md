# Grid Quest: A Trilha das Áreas 🦉📐

Um jogo educativo interativo focado no ensino de geometria (área e perímetro) com estética **Neobrutalista Moderna** e mecânicas inspiradas no Duolingo.

## 🚀 Sobre o Projeto
O **Grid Quest** foi desenvolvido para transformar o aprendizado de matemática em uma jornada visual e gamificada. O jogo utiliza o conceito de "Unidades de Conhecimento" (Sprints) para levar o aluno desde a contagem básica de quadradinhos até o domínio de fórmulas complexas e conversões de escala.

### 🎨 Estética e UX
- **Neobrutalismo Moderno:** Cores vibrantes, bordas pretas grossas (shadow-offset), tipografia imponente e icons dinâmicos.
- **Micro-recompensa:** Sons de sucesso, vibrações (Haptic feedback) e animações de confete ao concluir desafios.
- **Mobile-First:** Interface totalmente responsiva, otimizada para celulares, tablets e Chromebooks.

## 🧠 Arquitetura Pedagógica
A trilha é dividida em unidades fundamentais:
1.  **O Despertar do Quadrado:** Construção da intuição figural e contagem.
2.  **O Mestre Ladrilhador:** Transição para o pensamento multiplicativo ($A = b \times h$).
3.  **Dimensão Real:** Diferenciação entre Área e Perímetro e introdução de unidades de medida ($m^2$, $cm^2$, $km^2$).

## 🛠️ Tecnologias Utilizadas
- **Core:** Vanilla JavaScript (ES6+).
- **Styling:** CSS Grid, Flexbox e Tailwind CSS (via CDN para rapidez).
- **Icons:** Google Material Symbols.
- **Fonts:** Plus Jakarta Sans & Lexend (via Google Fonts).

## 📂 Estrutura de Arquivos
- `index.html`: Menu inicial e trilha de progresso.
- `game.html`: Shell do motor de jogo.
- `css/style.css`: Design System e animações neobrutalistas.
- `js/engine.js`: O "cérebro" do jogo (lógica de validação e física).
- `js/data.js`: Banco de dados de desafios pedagógicos.

## 🏁 Como Rodar Localmente
Basta abrir o arquivo `index.html` em qualquer navegador moderno. Não requer servidor backend ou instalação de dependências.
